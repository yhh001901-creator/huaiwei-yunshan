# 淮味云膳餐厅服务系统 - 功能文档

## 1. 系统概述

淮味云膳是一个完整的餐厅服务系统，采用前后端分离架构，包含Web管理端、Web客户端、移动端应用等多个终端。系统提供菜品管理、订单管理、会员管理、优惠券管理等多种功能，并集成了AI推荐功能。

## 6. 图片上传功能

### 6.1 上传接口说明
- 后端实现文件上传
- 静态资源访问配置
- 文件大小限制
- 文件格式支持

### 6.2 图片存储结构
```
uploads/
├── dish image/        # 菜品图片（分类存储）
│   ├── 主食/
│   ├── 淮扬菜/
│   ├── 甜品/
│   └── 饮料/
└── dishes/            # 上传的菜品图片
```

### 6.3 文件命名规则
- 格式：`dish_{timestamp}_{original_filename}`
- 示例：`dish_1234567890_红烧肉.png`

## 8. 系统配置

### 8.1 后端配置
- 服务端口：8083
- 数据库：MySQL 8.0
- 文件上传路径：uploads/dishes/
- 最大文件大小：10MB

### 8.2 前端配置
- 开发端口：5173
- 后端API地址：http://localhost:8083/api
